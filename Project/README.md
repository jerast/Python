# Proyecto Data Mining con SQL Server

## Descripción

Este proyecto tiene como objetivo emplear técnicas de minería de datos y aprendizaje automático para analizar y predecir el comportamiento de ventas de motocicletas Yamaha en el próximo año. Utilizaremos un conjunto de datos históricos que abarca un período de cinco años para entrenar modelos de clasificación y generar predicciones precisas.

## Pre-requisitos

Para correr el proyecto, **necesitas tener instalado**:

- python
- python-venv
- SQL Server ODBC driver

## Instalación

1. Ve al directorio del proyecto ejecutando `cd Project` desde la terminal.
2. Crea un entorno virtual de Python, por ejemplo `python3 -m venv project_env`.
3. Activa el entorno virtual con `source project_env/bin/activate`.
4. Instala los paquetes y dependencias con `pip install -r requirements.txt`.
5. Ejecuta `python3 main.py` para iniciar el proyecto.
